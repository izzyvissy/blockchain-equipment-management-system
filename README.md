# EMS Blockchain System

Blockchain-based Equipment Management System (EMS) using Solidity, Hardhat, React, and Polygon Amoy Testnet.

---

# Project Overview

EMS is a decentralized application (DApp) designed to manage sports equipment borrowing and returning using blockchain technology.

The system uses:
- Smart Contracts for immutable transaction logic
- MetaMask for wallet authentication
- Polygon Amoy Testnet for blockchain deployment
- ReactJS frontend for user interaction

Core roles:
- Administrator
- Storekeeper
- User (Student)
